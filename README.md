# onix - 一个操作系统的设计和实现

[本文档地址](https://github.com/StevenBaby/onix)

原系统用 C 实现，计划用 C++ 重写。

## 开发环境

---

### bochs-gdb

在 `build/bochs-gdb` 目录执行如下命令，安装 `bochs-gdb`，只针对 Archlinux。

```sh
makepkg -si
```
